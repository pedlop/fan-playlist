A EQUIPE ASBUILT

Desenvolvendo Aplicações!

ASBuilT - Application and Systems Builder Team.

A ideia de ter uma equipe para desenvolvimento de aplicações, tanto web quanto mobile, teve início após a realização de diversos trabalhos pelos integrantes da equipe, juntamente com a ideia que um de nossos membros teve quanto ao desenvolvimento de uma aplicação relacionada a Shows de Músicas, logo veio também a criação do nome da equipe que iria desenvolvê-la. Todos os membros estão no 7º período do curso de Engenharia de Software pela Universidade Federal de Goiás. A criação de um website desta aplicação veio com a necessidade de construir uma ferramenta web para a disciplina Desenvolvimento de Software para Web 2017/1, ministrada pelo professor Walison Moreira. Toda a documentação está disponível apenas para os membros da equipe na solução Git conhecida com Bitbucket. O repositório deste website pode ser encontrado no Github em FanPlaylist.

TERMOS DE USO DA APLICAÇÃO

FAN, ESCOLHA SUAS MÚSICAS PREDILETAS!

A equipe ASBuilT criou este site para os fãs de música. Só poderá ser utilizado um e-mail por usuário, e o voto será contabilizado para cada cantor do seu show escolhido. Caso queira compartilhar algum show com seus amigos nas redes sociais basta realizr o Login pelo Facebook ou Google e compartilhar na sua linha do tempo. O resultado final da votação das músicas será divulgado em uma data previamente disponibilizada no próprio site. Seus dados utilizados são os básicos de Perfil, selecionados no cadastro ou por meio da rede social (Nome, e-mail, foto).
